Projekt serwera Raspberry Pi z dodatkowymi funkcjonalnościami.

Ukończone Lekcje

Lekcja 1

Podstawowy Dashboard i Kamera

Flask app z metrykami systemowymi (CPU, RAM, temperatura). Kamera HD z libcamera-vid, autoryzacja, galeria zdjęć.

Lekcja 2

Cloudflare Tunnel i Dostęp Zdalny

Dostęp globalny przez tunel, SSH terminal w przeglądarce. Zero Trust security, pierwszy backup system.

Lekcja 3

System Alertów Multi-Channel

Powiadomienia email, Discord, Telegram. Progi temperaturowe i CPU. Cooldown 5 minut.

Lekcja 4

Enterprise Backup Strategy

fsarchiver - pełne obrazy partycji, kompresja zstd, automatyczny upload do Google Drive. Czas wykonania: 5 minut.

Lekcja 5

Kontrola Wersji i GitHub

Repozytorium Git, .gitignore, uwierzytelnienie SSH keys. Publiczne repo na GitHub.

Lekcja 6

Production Deploy Stack

Migracja do /var/www/, production stack. 3 workers Gunicorn, proxy przez Nginx. Systemd services.

Lekcja 7

Multi-Service Architecture

files.gingerity.space, FileBrowser na porcie 8081. Rozwiązanie konfliktów SSH vs SFTP routing.

Lekcja 8

Performance i Threading

ThreadPoolExecutor, asynchroniczne operacje kamery, eliminacja blocking calls. Poprawki security validation i error handling.

Lekcja 9

Modern Frontend Design

CSS Variables, Inter font, flexbox layout. Responsive design bez scroll na desktop. Glass-morphism effects.

Status

System Produkcyjny

Nginx+Gunicorn, multi-service architecture, async operations, disaster recovery. Działający system IoT.
