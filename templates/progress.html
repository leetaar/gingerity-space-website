PROJECT PROGRESS

GINGERITY.space

Projekt serwera Raspberry Pi z dodatkowymi funkcjonalnościami.
Linux, webdev, networking.

Ukończone Lekcje

Lekcja 1

Podstawowy Dashboard i Kamera

Flask app z metrykami systemowymi (CPU, RAM, temperatura). Kamera HD z libcamera-vid, autoryzacja, galeria zdjęć.

Lekcja 2

Cloudflare Tunnel i Dostęp Zdalny

Dostęp globalny przez tunel, SSH terminal w przeglądarce. Zero Trust security, pierwszy backup system.

Lekcja 3

System Alertów Multi-Channel

Powiadomienia email, Discord, Telegram. Progi temperaturowe i CPU. Cooldown 5 minut.

Lekcja 4

Enterprise Backup Strategy

fsarchiver - pełne obrazy partycji, kompresja zstd, automatyczny upload do Google Drive. Czas wykonania: 5 minut.

Lekcja 5

Kontrola Wersji i GitHub

Repozytorium Git, .gitignore, uwierzytelnienie SSH keys. Publiczne repo na GitHub.

Lekcja 6

Production Deploy Stack

Migracja do /var/www/, production stack. 3 workers Gunicorn, proxy przez Nginx. Systemd services.

Lekcja 7

Multi-Service Architecture

files.gingerity.space, FileBrowser na porcie 8081. Rozwiązanie konfliktów SSH vs SFTP routing.

Lekcja 8

Performance i Threading

ThreadPoolExecutor, asynchroniczne operacje kamery, eliminacja blocking calls. Poprawki security validation i error handling.

Lekcja 9

Modern Frontend Design

CSS Variables, Inter font, flexbox layout. Responsive design bez scroll na desktop. Glass-morphism effects.

Lekcja 10

Stabilizacja Systemu i Debugging

Zaawansowana diagnostyka i rozwiązywanie problemów systemowych. Ustabilizowanie Gunicorn, Nginx i procesów aplikacji. Poprawna konfiguracja workerów.

Planowane Lekcje

Lekcja 11: Wzmocnienie Bezpieczeństwa (Hardening)

Konfiguracja firewalla UFW i wdrożenie Fail2Ban w celu zabezpieczenia serwera przed atakami i nieautoryzowanym dostępem.

Technologie: UFW, Fail2Ban

Lekcja 12-13: Konteneryzacja (Docker)

Spakowanie wszystkich usług do kontenerów i zarządzanie całym stosem za pomocą jednego pliku konfiguracyjnego.

Technologie: Docker, Docker Compose

Lekcja 14: Profesjonalny Monitoring

Zastąpienie obecnych wykresów stosem Prometheus + Grafana. Budowa zaawansowanego, interaktywnego dashboardu.

Technologie: Prometheus, Grafana

Lekcja 15: Zaawansowane Usługi Sieciowe

Wdrożenie serwera DNS Pi-hole do ochrony sieci oraz konfiguracja prywatnej sieci VPN za pomocą WireGuard.

Technologie: Pi-hole, WireGuard

Lekcja 16: Prywatna Chmura Plików (NAS)

Uruchomienie usługi Samba w kontenerze. Udostępnienie dysku SSD jako dysku sieciowego (mapowanego) dla komputerów w sieci lokalnej.

Technologie: Samba, Docker

Lekcja 17: Własny Serwer Pocztowy

Uruchomienie w pełni funkcjonalnego serwera pocztowego. Konfiguracja rekordów DNS (MX, SPF, DKIM) dla zapewnienia dostarczalności.

Technologie: Docker Mailserver

Etap Ekspercki

Orkiestracja i Zarządzanie Sekretami

Wprowadzenie do zarządzania kontenerami przez interfejs graficzny Portainer. Implementacja bezpiecznego przechowywania haseł i kluczy API za pomocą Vault.

Cel: Bezpieczeństwo i zarządzanie na poziomie korporacyjnym.
